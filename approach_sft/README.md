## How to run:

```bash
python launch.py <experiment_name>
```

## Supported experiments 
See `config/experiment/`, but the following ones are the main ones:

   - `answer_only_arithmetic`
   - `answer_only_gsm8k`
   - `cot_arithmetic`
   - `cot_gsm8k`
